# 🏥 Health Prediction Application

## Project Overview

This project is a Health Prediction Application developed using Machine Learning, Streamlit, and SQLite. The application predicts diabetes risk based on medical parameters such as Pregnancies, Glucose level, Blood Pressure, Skin Thickness, Insulin, BMI, and Diabetes Pedigree Function. Users can enter patient details through an interactive interface, and the trained machine learning model generates predictions indicating whether the patient has a high or low diabetes risk.

## Dataset Description

The dataset used in this project contains medical details of different patients and helps in predicting diabetes risk. It includes features like glucose level, blood pressure, insulin, BMI, pregnancies, and diabetes pedigree function. Each row represents a patient’s health record, and the machine learning model studies these patterns to predict whether the person is likely to have diabetes or not. This dataset is very useful for understanding how AI and machine learning can be applied in the healthcare field.

## Features of the Application

* Add new patient records
* View saved patient records
* Update patient details
* Delete patient records
* Predict diabetes risk using machine learning

## Technologies Used

* Python
* Streamlit
* Pandas
* SQLite
* Scikit-learn
* NumPy
* Pickle

## API Usage

In this project, an API concept is used through the Streamlit web application, where user inputs are sent to the machine learning model and predictions are generated in real time. Although no external third-party API is integrated, the application functions similarly to a prediction API by accepting health data and returning diabetes risk results instantly.

## Deployment

The deployment part of this project was done using Streamlit, which helped in creating a simple and interactive web application for the machine learning model. The application can be deployed online using platforms like Streamlit Cloud, allowing users to access it through a web browser and get real-time predictions easily.

## Conclusion

This project demonstrates how machine learning can be integrated with web applications to create practical healthcare solutions. It shows the importance of AI in predicting diseases early and managing patient records efficiently.
